# EarnFlow – Crypto Trading & Earnings Platform

A fully functional front-end web application that simulates crypto trading (BTC/USDT) with a virtual wallet, reward points, deposit requests via IBAN, and a complete user dashboard.

![Screenshot](screenshots/home.png)

## 🚀 Features

- **User Authentication** – Sign up / Login with email and phone validation.
- **Live Trading Simulation** – Buy/Sell BTC using USD amounts; real-time price updates (simulated).
- **Reward Points*S* – Earn 1 point for every trade (buy or sell).
- **Virtual Wallet** – Demo account starts with $50,000 virtual balance (withdrawals disabled for demo).
- **Deposit System** – View IBAN, submit deposit requests (manual confirmation by admin, 24-36h).
- **P&L Tracking** – Total, Today's, and Loss Recovered metrics.
- **Surveys** – Browse and start sample surveys.
- **Responsive UI** – Works on desktop and mobile.

## 🛠️ Technologies Used

- HTML5, CSS3, JavaScript (vanilla)
- Font Awesome for icons
- TradingView widget for chart (loaded via CDN)
- localStorage for data persistence (no backend required)

## 📦 How to Run

1. Clone this repository.
2. Open `index.html` in your browser.
3. Use the demo account: `demo@earn.com` / `123456` or create your own.

## 📝 Notes

- All data is stored in your browser's `localStorage`.
- The deposit system is a simulation – no real money is involved.
- Withdrawals are disabled for the demo account.
